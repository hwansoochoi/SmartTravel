01. 통합 운영 인프라

Inquiry → Alert → Database → Field Reporting

핵심 관리 도구 셋업

Input: 구글 폼 (Google Forms)

한국 여행사의 모든 의뢰를 정형화된 양식으로 접수합니다. 재질의가 필요 없는 '필수 체크리스트' 기반의 문항 설계로 정보 누락을 원천 차단합니다.

Notification: 슬랙 (Slack)

의뢰 접수부터 현지 수배, 가이드 보고까지 모든 소통의 허브입니다. Zapier 연동을 통해 실시간 알림 시스템을 구축하여 즉각적인 대응을 지원합니다.

Database: 구글 시트 (Sheets)

모든 프로젝트 히스토리가 자동으로 누적되는 마스터 DB입니다. 필터링과 검색을 통해 언제든 과거 견적과 수배 내역을 1초 만에 조회할 수 있습니다.

Storage: 구글 드라이브 (Drive)

최종 견적서, 바우처 등 모든 파일을 링크 형태로 관리합니다. 카카오톡의 파일 만료 문제에서 완전히 벗어나 영구적인 문서 자산화를 실현합니다.

표준 업무 워크플로우

의뢰 (Inquiry)

여행사 담당자가 공지된 구글 폼으로 상세 수배 의뢰

수배 (Search)

슬랙 알림 확인 후 현지 중계사/업체와 가부 확인 및 견적 산출

확정 (Execution)

최종 바우처 발행 및 현지 가이드 배정 및 오리엔테이션

정산 (Settlement)

행사 종료 후 가이드 정산 및 결제 당일 환율 기준 입금 확인

Step 1: 정보의 규격화 (Input)

스마트 접수 시스템

  • 표준 양식 제공: 구글 폼을 활용하여 여행사별 산발적인 의뢰를 하나의 규격화된 포맷으로 통일합니다.
  • 접근성 극대화: 거래처별 슬랙(Slack) 채널 상단 북마크에 폼 링크를 고정하여 언제든 즉시 의뢰 가능합니다.
  • 휴먼 에러 차단: 재질의가 필요 없도록 설계된 필수 문항을 통해 커뮤니케이션 비용을 최소화합니다.

필수 데이터 추출 및 연동

  • 핵심 수집 항목: 행사 성격(연수/VIP/FIT), 상세 일정, 인원, 확정 항공편, 인당 예산을 필수로 수집합니다.
  • 즉시 DB화: 제출과 동시에 구글 시트 마스터 DB에 행(Row) 단위로 자동 기록되어 수기 입력이 불필요합니다.
  • 자동화 트리거: 폼 제출 완료 즉시 Zapier가 이를 감지하여 '실시간 알림' 워크플로우를 작동시킵니다.

Step 2: 실시간 대응 (Notification)

실시간 슬랙 알림

Zapier를 통해 구글 폼 응답을 슬랙 카드 형태로 즉시 전송합니다. 팀원 모두가 동시에 의뢰를 확인합니다.

스레드(Thread) 관리

개별 행사 관련 대화는 해당 알림 메시지의 '댓글'에서만 진행하여 대화 흐름의 파편화를 방지합니다.

이모지 승인

확인(👀), 진행중(⏳), 완료(✅) 이모지를 사용하여 불필요한 텍스트 답변 없이 업무 상태를 공유합니다.

Step 3: 데이터 마스터 (DB)

항목 (DB Columns) 설명 비고
Project ID YYMMDD-여행사명-지역 (예: 260710-OO투어-오사카) 고유 식별자
진행 상태 의뢰접수 / 수배중 / 견적발송 / 확정 / 정산완료 드롭다운 관리
최종 바우처 링크 구글 드라이브 내 해당 파일 고유 URL 영구 보존
환율 적용가 송금일 기준 TTS 환율 자동 계산 수식 적용 정산 정확도

02. 실전 업무 시나리오

5인 미만 소정예 팀의 현장 대응 가이드

Scenario A: 신규 수배 및 견적

1단계: 수배 착수 및 공유

슬랙 알림 확인 후 담당자가 '👀' 반응을 남깁니다. 일본 중계사에게 가부 확인 메일을 보내고 슬랙 스레드에 "호텔 확인 중 - 15시 회신 예정"이라고 남깁니다.

2단계: 견적서 및 리스크 관리

확인된 금액을 바탕으로 '표준 견적서'를 발행합니다. 하단에 유효기간, 불포함 사항, 취소 규정을 명시하여 법적/금전적 리스크를 사전에 차단합니다.

Scenario B: 현장 케어 및 보고

100%
현지 가시성 확보

가이드 실시간 보고 매뉴얼

  • 미팅 보고: 가이드 성함/연락처, 차량 번호, 미팅 완료 여부를 슬랙 #03_현장_케어 채널에 즉시 업로드.
  • 상황 보고: 현지 날씨, 도로 정체, 일정 변경 이슈 발생 시 즉각 스레드에 멘션(@팀장)하여 실시간 모니터링.

전문가급 유지관리 팁

  • 파일 만료 방지: 모든 최종 견적서와 바우처는 슬랙에 직접 올리지 않고, 구글 드라이브 폴더 링크로만 공유하여 영구 보존합니다.
  • 검색의 생활화: 슬랙의 검색 기능(`has:link`, `from:@가이드`)을 활용하여 과거의 유사 수배 사례와 견적 데이터를 5초 내에 찾아 활용합니다.
  • 예약 메시지 활용: 가이드에게 전달할 익일 지시 사항은 슬랙의 '메시지 예약' 기능을 사용하여 매너 있는 시간에 자동 발송되도록 설정합니다.
  • 권한 관리: 외부 중계사는 슬랙의 '멀티 채널 게스트'로 초대하여 보안을 유지하고 소통 창구를 일원화합니다.
